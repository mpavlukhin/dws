Depixelizer Web Service

WELCOME TO THE DEPIXELIZER WEB SERVICE!

Just choose option you want and get started!

Improve your picture quality Improve your picture quality (neural network) Improve your web camera

The first option is "Improve your picture".
It is based on OpenCV (Open Computer Vision) library.
Depixelizer Web Service uses library's technologies (smoothing, filters, etc.) to improve quality of your picture.
This method of image processing is extremely fast and qualitative.

The second option is "Improve your picture quality (neural network)".
It is based on CNN (Convolutional Neural Network).
Depixelizer Web Service is Machine Learning.
It learns to improve your pictures quality!
Awesome thing, isn't it?
This method of image processing is non-extremely fast but extremely qualitative.

The third option is "Improve your web camera".
It is also based on OpenCV and it uses library's technologies (smoothing, filters, etc.) to improve quality of your picture.
This function allows you use your web camera with improved image quality.
This method of image processing is extremely fast and qualitative.
